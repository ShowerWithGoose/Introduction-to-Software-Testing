Problem Description: Write a program to convert a floating-point number represented in decimal form into scientific notation. The input data contains no sign, has digits before and after the decimal point, and all digits are significant (i.e., there are no trailing zeros after the decimal point). If there’s only one digit before the decimal point, it can be 0; otherwise, the first digit before the decimal point cannot be 0.

Input Format: A decimal number is input from the console, ending with a newline character.

Output Format: The scientific notation of the input number, consisting of the following parts:
1. The base is either a decimal or an integer. If it’s a decimal, it must have one significant digit before and after the decimal point (i.e., the digit before the decimal point is non-zero, and there are no trailing zeros after the decimal point). If it’s an integer, it contains only one significant digit without a decimal point.
2. The letter “e” in lowercase.
3. The exponent, which is an integer. If it’s non-negative, it appears without the + sign. If it’s negative, it is prefixed with a - sign, and the first digit of the integer part is non-zero.
