Problem Description: Input a string containing the expansion symbol '-' from the keyboard, and expand it into its equivalent full characters. For example, expand a-d to abcd, and output the expanded string.

Requirements:
Only process characters within the ranges [a-z], [A-Z], and [0-9].
Perform expansion only when both characters around the expansion symbol are of the same type (either lowercase letters, uppercase letters, or digits), and the character after the expansion symbol is greater than the one before it.
For other cases (e.g., a-R, D-e, 0-b, 4-B), no expansion is performed, and the input is output unchanged.

Input Format: A string containing the expansion symbol '-' is input from the keyboard.

Output Format: Output the expanded string.
