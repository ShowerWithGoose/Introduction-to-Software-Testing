Problem Description: Given a sequence of integers from standard input, construct a binary search tree (BST) where the value of the left child is less than the value of the root, and the value of the right child is greater than or equal to the value of the root. Then, from left to right, output the values and heights of all the leaf nodes (the height of the root node is 1).

Input Format: First, read an integer `n` representing the number of integers. Then, read `n` integers in the next line, separated by spaces.

Output Format: For each leaf node, output its value and height in separate lines, in left-to-right order. The value and height should be separated by a space.
