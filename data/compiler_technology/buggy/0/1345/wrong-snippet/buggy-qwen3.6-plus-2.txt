**Summary of the Issue:**
According to the problem description, when the symbols `&` or `|` appear individually, the program should detect this as a lexical error and output the error category code `a`. However, in the provided code, the `writeError()` method is called with `Character.toString(c)`, which passes the actual symbol (`"&"` or `"|"`) instead of the required error code `"a"`. This causes the error output file to contain incorrect error type codes.
