Based on the problem description and code review, the main issue appears to be in the `parseCompUnit` method in Parser.java. The method doesn't properly handle the loop structure for parsing multiple declarations and function definitions before the main function. It only adds the last parsed node to the children list instead of all parsed nodes.

    public Node parseCompUnit() {
        ArrayList<Node> children = new ArrayList<>();
        Node node = null;
        while (true) {
            if (curToken == null) {
                break;
            }
            /* root node*/
            if (tokens.preRead(1).getType() == TokenType.MAINTK) {
                node = parseMainFunDef();
            } else if (tokens.preRead(2).getType() == TokenType.LPARENT) {
                node = parseFuncDef();
            } else if (curToken.getType() == TokenType.CONSTTK) {
                node = parseConstDecl();
            } else if (curToken.getType() == TokenType.INTTK || curToken.getType() == TokenType.CHARTK) {
                node = parseVarDecl();
            } else {
                break;
            }
            children.add(node); // This line should be inside the loop
        }
        // children.add(node); // This line should be removed
        return new Node(children, SyntaxType.CompUnit);
    }
