Based on the problem description and code review, the main issue appears to be in the CompUnit parsing logic in Parser.java. The parser doesn't properly handle the case where there are multiple declarations and function definitions before the main function, and the loop condition `while(i <= symList.size())` can cause an IndexOutOfBoundsException. Additionally, the parser doesn't correctly advance through tokens when parsing different components.
