The likely error is in the `getNextToken()` method in `Syntax.java`, where it does not handle the case when `lexer.nextToken()` returns `null`, leading to a `NullPointerException` when `nowToken` is used later.

    public Token getNextToken() {
        pos++;
        if (pos < tokens.size()) {
            nowToken = tokens.get(pos);
            return nowToken;
        }
        while (true) {
            Token token = lexer.nextToken();
            nowToken = token;
            tokens.add(token);
            break;
        }
        return nowToken;
    }
