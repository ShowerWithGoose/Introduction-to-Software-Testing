Based on the problem description and code review, the main issue appears to be in the `CompileUnit.java` file where the parsing logic doesn't properly handle the grammar structure. The parser incorrectly assumes that declarations must come before function definitions, but according to the grammar `CompUnit → {Decl} {FuncDef} MainFuncDef`, it should parse all possible declarations first, then all possible function definitions, and finally the main function.
