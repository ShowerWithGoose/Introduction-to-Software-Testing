Based on the problem description and code review, the main issue appears to be in the CompUnit parsing logic in Parser.java. The current implementation has complex and potentially incorrect lookahead logic for distinguishing between Decl, FuncDef, and MainFuncDef. The grammar specifies CompUnit → {Decl} {FuncDef} MainFuncDef, but the current code tries to handle overlapping cases in a way that may not correctly parse valid programs or detect errors.
