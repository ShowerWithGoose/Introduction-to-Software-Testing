Based on the problem description and code review, I can see that the test case input shows a compilation error where `javac` is not found, but this appears to be an environment issue rather than a code bug. However, looking at the actual source code, there's a clear bug in the Lexer's handling of single quotes and ampersand escaping in character constants.
