Problem Description: Write a program to multiply two univariate polynomials.

Input Format: First, input the coefficients and exponents of non-zero terms from the first polynomial, separated by a space. The exponents of the polynomial terms are all non-negative integers, and the coefficients and the maximum degree of the polynomial will not exceed the range of the `int` type.  
For a polynomial in the form \(a_nx^n + a_{n-1}x^{n-1} + \ldots + a_1x^1 + a_0x^0\), the input format is as follows:  
a_n n a_{n-1} n-1 ... a1 1 a0 0  
Each pair of integers represents the coefficient and exponent of a term, respectively. Only terms with non-zero coefficients will appear in the input. There is no space after the last exponent, and the input ends with a newline character.  
Then, input the second polynomial in the same format.

Output Format: Output the result of the multiplication to the screen. Print the terms in descending order of exponents, with each termâ€™s coefficient and exponent separated by a space. A space after the last exponent is optional.
