Problem Description: Read an arithmetic expression from standard input, such as 5 - 1 * 2 * 3 + 12 / 2 / 2 =. Calculate the result of the expression and output it.
Requirements
1. The expression includes only the operators +, -, *, and /. The = symbol at the end of the expression indicates the end of input. The expression may contain spaces.
2. The expression does not contain parentheses, and is always valid.
3. For division /, perform integer division. For example, 5 / 3 should yield 1.

Input Format:Enter an arithmetic expression ending with `=` in the console.

Output Format:Output the integer result to the console.
