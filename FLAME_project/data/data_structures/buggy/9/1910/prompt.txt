Problem Description: In a 2D plane, each line segment is defined by two endpoints (each with coordinates (x, y)). If two line segments share an endpoint, they can form a continuous line segment. Assume each line segment has endpoints  v_1(x_1, y_1)  and  v_2(x_2, y_2)  with  x_1 < x_2 ; in this case,  v_1  is the starting point and  v_2  is the endpoint. For a continuous line segment, the  x -coordinates of its endpoints must be in increasing order  (x_i < x_{i+1})  for  i = 1, \dots, n-1 , where  n  is the number of endpoints in the continuous line segment.
Given a set of line segments (defined by the  x  and  y  coordinates of their endpoints), write a program to find the continuous line segment containing the maximum number of segments and output the number of segments in this continuous line and the coordinates of its starting point.
Constraints:
1. No two line segments intersect in the middle.
2. No three or more segments share a single endpoint.
3. Only one continuous line segment contains the maximum number of segments.

Input Format:
- The first line contains the number of line segments.
- Each following line contains the  x  and  y  coordinates of two endpoints for each segment. The first endpoint’s  x -coordinate is smaller than the second endpoint’s  x -coordinate, i.e.,  x_1 < x_2 . All coordinates are integers, and each pair of coordinates is separated by a space.

Output Format:
- Print the maximum number of segments in the longest continuous line segment, followed by the  x  and  y  coordinates of the starting point, separated by spaces.
