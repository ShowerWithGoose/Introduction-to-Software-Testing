Based on the problem description and test case, the program should output lexical tokens to `lexer.txt` but produces no output file. The issue appears to be in the Lexer class where character constants and string constants are not being handled correctly according to the token type specifications.

Looking at the expected output, we need:
- CharConst tokens to have category code CHRCON
- StringConst tokens to have category code STRCON

However, in the Lexer code, I notice that:
1. The `scanString()` method uses `TokenType.FString` which maps to "STRCON" (this seems correct)
2. There's no `scanChar()` method to handle character literals like `'a'` or `'\0'`
3. Character literals are likely being processed as identifiers or causing errors

The main issue is that character constants (like `'a'`, `'\0'`) are not being lexed properly. The lexer needs to handle single quotes for character constants.

    public Lexer lex() throws IOException {
        int chr;
        while ((chr = reader.read()) != -1) {
            if (Character.isWhitespace(chr)) {
                continue;
            }
            reader.unread(chr);

            if (Character.isDigit(chr)) {
                scanNumber();
            } else if (Character.isLetter(chr) || chr == '_') {
                scanIdentifier();
            } else if (chr == '"') {
                scanString();
            } else if (chr == '/') {
                scanSlash();
            } else {
                scanSymbol();
            }
        }
        return this;
    }

    private void scanString() throws IOException {
        int chr = reader.read();
        assert chr == '"' : "Invariant broken";
        Pair<Integer, Integer> start = reader.location();
        StringBuilder str = new StringBuilder();
        while ((chr = reader.read()) != '"') {
            if (chr == '%') {
                chr = reader.read();
                if (chr != 'd') {
                    errors.add(ErrorType.InvalidFormatString, location());
                    continue;
                }
                str.append('%').append('d');
            } else if (chr == '\\') {
                chr = reader.read();
                if (chr != 'n') {
                    reader.unread(chr);
                    errors.add(ErrorType.InvalidFormatString, location());
                    continue;
                }
                str.append('\n');
            } else if (chr != (char) 32 && chr != (char) 33 && (chr < (char) 40 || chr > (char) 126)) {
                errors.add(ErrorType.InvalidFormatString, location());
            } else {
                str.append((char) chr);
            }
        }
        addToken(TokenType.FString, '"' + str.toString() + '"', start);
    }
