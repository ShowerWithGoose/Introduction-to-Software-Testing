Based on the test case and expected vs actual output, the issue is that the parser is missing error detection for a missing closing parenthesis in function calls. The test case shows three locations where ')' is missing (lines 1, 3, and 5), but the parser only reports errors for lines 1 and 5.

Looking at the code, the problem appears to be in the `parseFuncDef` method in the Parser.java file. When parsing function parameters, it checks for the closing parenthesis, but there's a logic issue in how it handles the case when parameters are present.
