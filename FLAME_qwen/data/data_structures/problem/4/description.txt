Problem Description: Write a program to count the occurrence of each word (word frequency) in an English text file and output the results to the screen in lexicographical order.

Notes:
A "word" is defined as a sequence of characters consisting only of letters.
Words containing uppercase letters should be converted to lowercase before counting.

Input Format: The program reads English words from the file named "article.txt" located in the current directory.

Output Format: The program outputs the word frequency results to the screen in lexicographical order, with one word and its occurrence count per line. Each line consists of a word followed by a space and its count. There should be no extra spaces after the count, and each line ends with a newline.
