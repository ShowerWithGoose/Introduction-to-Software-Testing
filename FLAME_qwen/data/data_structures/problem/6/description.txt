Problem Description: Assume that the initial state of the given integer stack is empty, and the stack has a maximum capacity of 100. A sequence of stack operations is provided as input. You need to output the sequence of elements popped from the stack in the order of the operations. 

Stack operations:
- **1** followed by an integer (which is not 1, 0, or -1) represents a push operation with the integer being pushed onto the stack.
- **0** represents a pop operation.
- **-1** represents the end of operations.

Input Format: A sequence of stack operations is given as input, where integers representing the values to be pushed onto the stack and the integers representing the stack operations are separated by a space.

Output Format: On a single line, output the sequence of elements popped from the stack in the order of the operations, separated by spaces. Ensure there is a space after the last element. If a pop operation is attempted when the stack is empty, or if a push operation is attempted when the stack is full, output the string "error" followed by a space. After all operations have been performed, the stack may not be empty.
