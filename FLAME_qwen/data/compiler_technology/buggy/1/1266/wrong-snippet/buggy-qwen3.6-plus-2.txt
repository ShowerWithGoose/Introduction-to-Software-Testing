However, examining the provided source code reveals that it heavily utilizes **modern Java features** that will cause a `Compilation Error` if the grading environment uses an older Java version (e.g., Java 8 or 11):

2. **Pattern Matching for `instanceof`** (`if (obj instanceof Type var)`) is used extensively in `Validator.java` (requires Java 16+).
