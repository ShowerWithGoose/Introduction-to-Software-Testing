1. **Java Version Incompatibility**: `Parser.java` uses the switch expression arrow syntax (`case IDENFR -> { ... }`). This syntax was introduced in **Java 14**. If the grading/compilation environment uses Java 8 or 11 (which is common in many academic settings), this will directly cause a `Compilation Error`.
