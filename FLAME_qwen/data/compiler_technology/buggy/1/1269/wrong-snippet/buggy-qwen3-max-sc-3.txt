Based on the problem description and code review, the main issue appears to be in the **CompUnit()** method in Parser.java. The method has infinite while loops that don't properly advance the lexer position when none of the conditions are met, causing the parser to get stuck. Additionally, the logic for determining whether to parse a FuncDef vs VarDecl is flawed.
