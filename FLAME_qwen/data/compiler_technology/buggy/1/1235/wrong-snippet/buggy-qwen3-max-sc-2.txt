Based on the problem description and code review, the main issue appears to be in the error handling implementation. The `Error` class uses a static list that isn't properly managed across instances, and the error reporting mechanism doesn't correctly accumulate errors. Specifically, in the `Lexer.java` file when handling single '&' or '|' characters, an error is added incorrectly by creating a new Error instance and calling addErrors on it, which doesn't actually add to the static list properly.
