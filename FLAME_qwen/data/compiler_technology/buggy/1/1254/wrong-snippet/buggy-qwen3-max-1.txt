Based on the problem description and code review, the main issue appears to be in the `parseDecls()` method in Parser.java. The method incorrectly checks for LPARENT token to determine when to stop parsing declarations, but according to the grammar, declarations should continue until we encounter either a function definition or main function definition. The current logic will break too early when encountering a '(' which could be part of other constructs.
