**Summary**
- **Problem Description**: Design a lexical analyzer that reads `testfile.txt`, identifies tokens (keywords, identifiers, operators, etc.), and outputs them to `lexer.txt`. If lexical errors occur (specifically error type `a` for standalone `&` or `|`), output the line number and error code to `error.txt`.
- **Test Case Input**: `/bin/sh: 1: javac: not found` (Indicates a missing Java compiler in the execution environment, but we will focus on logical/semantic flaws within the provided source code).
- **Identified Errors**:
  1. **Missing Bounds Checks**: In the `next()` method, when checking for two-character operators (`!`, `<`, `>`, `=`, `&`, `|`), the code directly calls `source.charAt(curPos)` without verifying `curPos < length`. This will throw a `StringIndexOutOfBoundsException` if these symbols appear at the end of the file.
  2. **Flawed `/* */` Comment Parsing**: The `while` loop condition `source.charAt(curPos) != '*'` causes the loop to exit immediately upon encountering `*`, failing to properly check for and consume the closing `/`. This leaves `*/` to be incorrectly parsed as separate `MULT` and `DIV` tokens in subsequent calls.
  3. **File Output Mode**: `FileOutputStream` is initialized with `true` (append mode). This causes outputs to accumulate across multiple program runs instead of overwriting the file, which will likely cause incorrect grading results.

    public void output() {
        if (errorList.isEmpty()) {
            File file = new File("lexer.txt");
            try {
                if (!file.exists()) {
                    Files.createFile(file.toPath());
                }
                try (FileOutputStream fos = new FileOutputStream(file, true)) {
                    for (Token t : tokenList) {
                        String str = t.getType().toString() + ' ' + t.getContent() + '\n';
                        fos.write(str.getBytes());
                    }
                }
            } catch (IOException e) {
                e.printStackTrace();
                throw new RuntimeException("Error while writing tokens to file", e);
            }
        } else {
            File file = new File("error.txt");
            try {
                if (!file.exists()) {
                    Files.createFile(file.toPath());
                }
                try (FileOutputStream fos = new FileOutputStream(file, true)) {
                    for (Error err : errorList) {
