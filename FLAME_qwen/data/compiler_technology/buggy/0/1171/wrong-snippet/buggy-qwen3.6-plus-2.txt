**Summary of the Issue:**
According to the problem description, the lexical analyzer should output token information to `lexer.txt` for correct programs, but must output error information to `error.txt` when lexical errors are detected. The provided `Compiler.java` unconditionally opens and writes to `lexer.txt` regardless of whether errors were found. This violates the output format specification and will cause the grading system to fail when testing erroneous inputs.
