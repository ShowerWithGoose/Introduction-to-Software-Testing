**Problem Description**

Please design and implement a lexical analysis program based on the grammar document. The program should be able to handle both correct and erroneous source programs.

* For a correct source program, the program should identify words from the source code, record their token type and value, and output the token type code along with the character or string representation of the token.

* For an erroneous source program, the program should detect errors and output the line number of the error along with the corresponding error type code.

In the lexical analysis process, the token type codes are uniformly defined as follows:

**Input Format**
The test program is provided in **testfile.txt**. It may be a correct source program or an erroneous source program.

**Output Format**
For a correct source program, output the word category code and the character/string form of the word in the order they are read into **lexer.txt**.
Format:
word category code character/string form of the word (separated by a single space)  

For an erroneous source program, output the line number where the error occurs and the error category code in ascending order of line numbers into **error.txt**. The line numbers start from 1.
Format:
error line number error category code (separated by a single space)


**Special Reminder**
Ensure that the read string is preserved exactly as it appears for output purposes, especially for numerical values.
In this assignment, each erroneous example will contain only lexical analysis errors. Each erroneous example will have exactly one error.
