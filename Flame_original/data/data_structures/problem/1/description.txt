Problem Description: Given an integer N (1<=N<= 10), generate all permutations of integers from 1 to N.

Input Format: An integer N is given as input.

Output Format: Output N! lines, where each line is a unique permutation of integers from 1 to N. Each integer in a line is separated by a space. The output lines should be sorted in lexicographical order, which means smaller numbers appear as early as possible in each permutation.
