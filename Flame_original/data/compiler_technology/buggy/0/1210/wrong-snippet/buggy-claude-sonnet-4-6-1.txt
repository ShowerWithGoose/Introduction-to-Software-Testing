## Summary

The test case failed with a **compilation error**: `/bin/sh: 1: javac: not found`. This means the Java compiler isn't available in the execution environment — the code never ran at all.

The root cause is that the project has no build configuration file (no `pom.xml`, `build.gradle`, `Makefile`, etc.), so the grading system can't locate or invoke `javac` to compile the `.java` files.

Beyond the build issue, there's also a **logical bug** in `ERROR.java`: error output is being written to `lexer.txt` instead of `error.txt`. The problem description clearly states errors should go to `error.txt`, but the `FileWriter` in `ERROR.print()` opens `lexer.txt`.
