### Summary
The lexical analyzer identifies keywords by checking if a token exists in the `reserveWords` list and then dynamically mapping it to a `LexType` enum value using `Enum.valueOf(LexType.class, token.toUpperCase() + "TK")`. The `reserveWords` list correctly includes `"continue"`, but the `LexType` enum is missing the corresponding `CONTINUETK` constant. When the lexer encounters the `continue` keyword in a source program, `Enum.valueOf` will throw an `IllegalArgumentException` at runtime, causing the program to crash. Additionally, the `LexType` enum should strictly match the token category codes defined in the problem description.
