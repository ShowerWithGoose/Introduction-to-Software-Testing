Based on the problem description and code review, I can see several issues in the parser implementation. The main problems appear to be:

1. In the `compUnit()` method, the logic for determining when to stop parsing declarations and function definitions is flawed
2. In the `stmt()` method, there's an error in the FOR loop parsing where it uses LBRACE instead of LPARENT
3. Several print methods in node classes have incorrect NodeType references

The most critical error is in the `stmt()` method where it incorrectly matches LBRACE for a for-loop instead of LPARENT.
